INTRODUCTION

on the surface, a database might seem much like a spreedsheet, it presents data arranged in columns and rows.
but that is where the similarity ends, because a database is far more powerful.

what can a database do?

A database can create relationship between tables that is, it is RELATIONAL.
Because of these relationship,a database supports complex querying,
with various combinations of columns across tables and filters to fine-tune
which rows return after the query executes.
